# PlatinumRx Data Analyst Assignment

## Overview

This repository contains solutions for the PlatinumRx Data Analyst Assignment. The project demonstrates practical skills in SQL, Spreadsheet analysis, and Python programming.

---

## Project Structure

* **SQL/**
  Contains database schema setup and query solutions for:

  * Hotel Management System
  * Clinic Management System

* **Spreadsheets/**
  Includes Excel workbook (`Ticket_Analysis.xlsx`) for:

  * Data lookup using VLOOKUP
  * Time-based analysis (same day & same hour tickets)

* **Python/**
  Contains scripts for:

  * Time conversion (minutes → hours & minutes)
  * Removing duplicate characters from a string

---

## Approach and Assumptions

* Used standard SQL syntax compatible with MySQL/PostgreSQL.
* Created schemas and queries based on assignment requirements.
* Added sample data to ensure queries run successfully.
* Used Excel formulas such as VLOOKUP, INT, and HOUR for analysis.
* Python scripts are written using basic logic and can be extended for edge cases.

2. Run:

